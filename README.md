# 자동차 시뮬레이션을 이용해 자율 주행 및 주차 로직 구현 (Self-Parking)

Python 기반의 시뮬레이터 환경에서 충돌 없이 지정된 주차 공간에 차량을 주차하는 경로 계획 및 제어 알고리즘 개발 프로젝트입니다.

기본 스켈레톤 및 시뮬레이션 코드 : [원본 저장소 링크](https://github.com/sungb0131/self-parking-sim)   
프로젝트 작업 파일 : self-parking-user-algorithms/**student_planner.py**

* 기간 : 2025.11.21 - 12.11
* 인원 : 손혜성, 왕민 (2명)
* **Role:**   
    왕민 : compute_path 함수 로직 전담 (경로 생성 함수)   
    손혜성 : compute_control 함수 로직 전담 (경로 추적 함수)

## 구현 과정
### 경로 생성 (compute_path) - 왕민

* Hybrid A* 알고리즘을 사용하여 '시작점 -> 목표점'과 '목표점 -> 시작점' 양쪽에서 동시에 경로를 탐색.
* 차량이 갈 수 있는 각도를 고려해 노드 탐색.
* 후진 주차가 필요할 때, 양쪽에서 시작한 노드가 만나는 접점을 기어 변경 지점으로 결정(변곡점)
* 최종 완성된 경로를 self.waypoints로 확정.


### 경로 추적 (compute_control) - 손혜성
 
* 전체 경로 중 차량과 가장 가까운 경로점을 찾아 현재 목표 지점으로 설정.
* 주행 방향에 따라 적절한 알고리즘 적용   
      * 전진 : Stanley Method   
      * 후진 : Pure Pursuit   
* 기어변경 지점 또는 최종 목표지점과의 잔여 거리를 기준으로 속도 제어.
